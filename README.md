# IW7 Dedicated Server Resources

This repository contains the latest IW7 Dedicated Server config files and scripts to easily start a server. [To get the server files and setup a IW7 Dedicated Server yourself, check out the official guide here.](https://docs.auroramod.dev/iw7-server-guide)


### iw7-mod/custom_scripts/

Inside this directory contains 3 folders for Zombies, Multiplayer, or both. Each folder has a blank GSC script that will load if the conditions are met and if code exists.
